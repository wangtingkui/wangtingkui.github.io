使用hugo搭建个人网站（三）- 引用其他文章

2019-12-02 1 分钟阅读 使用hugo搭建个人网站hugo

我们可以在文章中引用自己的其他文章，hugo 帮我们实现了这种功能，只需要在文章中使用 {{< ref "posts/my_post.md" >}}或者{{< relref "posts/my_post.md" >}}即可，前者是相对链接，后者是绝对链接

这个功能其实是 hugo 提供的 Shortcodes功能

一些例子：

1{{< ref "blog/post.md" >}} => https://example.com/blog/post/
2{{< ref "post.md#tldr" >}} => https://example.com/blog/post/#tldr:caffebad
3{{< relref "post.md" >}} => /blog/post/
4{{< relref "blog/post.md#tldr" >}} => /blog/post/#tldr:caffebad
5{{< ref "#tldr" >}} => #tldr:badcaffe
6{{< relref "#tldr" >}} => #tldr:badcaffe

如果是不是需要引用，而仅仅是需要展示{{< ref "blog/post.md" >}}这样的文本，需要在标签内加上/* */这样的注释，比如{{</* ref "blog/post.md" */>}}
